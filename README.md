# printf

> Recréation de `printf` — Projet 42

## Description

Implémentation de `ft_printf` avec gestion des conversions : `c`, `s`, `p`, `d`, `i`, `o`, `u`, `x`, `X`, `%`.

S'appuie sur une libft intégrée. Produit une bibliothèque statique `libftprintf.a`.

## Compilation

```bash
make
make re
make fclean
```

## Compétences

`C` · `Variadic functions` · `Makefile` · `Parsing de format`
